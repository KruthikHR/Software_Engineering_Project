# Bus and Tour Booking System

Software Engineering project developed using PHP and MySQL.

Features
- User registration and login
- Bus booking
- Tour package booking
- User dashboard
- Admin management

Technologies
- PHP
- MySQL
- HTML/CSS
- XAMPP

Database Setup

1. Open phpMyAdmin
2. Create a database named transport_booking
3. Import transport_booking.sql
4. Run the project using XAMPP
